# Vim Cheatsheet
Common commands to refer back to

## Buffers & Windows
|----------------|----------------|
|    Command     |        Purpose |
|    :----:      |           ---: |
|    <C-W><r>    | Rotate windows |
|----------------|----------------|
|    <C-W><r>    |    Swap window |
|                |    w/ the next |
|----------------|----------------|
| <C-W><Shift-H> |    Move window |
|                |    to the left |
|----------------|----------------|
| <C-W><Shift-K> |    Move window |
|                |    to the top  |
|----------------|----------------|
