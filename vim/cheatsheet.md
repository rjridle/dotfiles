# Vim Cheatsheet
Common commands to refer back to

## Buffers & Windows
|-------------|----------------|
|  Command    |        Purpose |
|   :----:    |           ---: |
|  <C-W><r>   | Rotate windows |
|-------------|----------------|
|  <C-W><x>   |    Swap window |
|             |    w/ the next |
|-------------|----------------|
| <C-W><Sh-H> |    Move window |
|             |    to the left |
|-------------|----------------|
| <C-W><Sh-K> |    Move window |
|             |    to the top  |
|-------------|----------------|
